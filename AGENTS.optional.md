# Optional AGENTS.md snippet

Add this to a repository-level AGENTS.md only if you want Codex to actively consider these skills during research tasks.

```markdown
## Research workflow skills

For ECE research tasks, prefer explicit skill invocation:

- Use `$research-triage-router` when the task is ambiguous.
- Use `$research-methodology-advisor` for methodology defense.
- Use `$research-data-troubleshooter` for data, convergence, or anomaly issues.
- Use `$critical-thinking-defense-coach` to stress-test claims.
- Use `$research-artifact-reviewer` for papers, thesis chapters, reports, and academic emails.
- Use `$research-code-simulation-reviewer` for research code and simulations.
- Use `$literature-search-citation-scout` for references, SOTA, standards, and citation validation.
- Use `$research-ethics-integrity-advisor` for integrity, authorship, plagiarism, and COI issues.
- Use `$research-progress-roadmap-planner` for milestones and research planning.

Do not fabricate citations, data, results, standards, or authorship claims. Ask for sources or use available search tools when evidence is required.
```
